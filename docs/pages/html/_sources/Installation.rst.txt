Pip install
################################

.. code-block:: console

    # Install from Pypi:
    pip install pypickle

    # Install directly from github
    pip install git+https://github.com/erdogant/pypickle


Uninstalling
################

.. code-block:: console

   # Remove the pypickle environment
   pip uninstall pypickle
